# 📊 E-Commerce Sales Analysis Dashboard

A complete Data Analytics Project built using SQL, Microsoft Excel, and Power BI  to analyze an E-Commerce sales dataset and generate actionable business insights through an interactive dashboard.

---

## 📌 Project Overview

This project focuses on analyzing an E-Commerce sales dataset containing customer orders, product information, payment methods, and sales transactions.

The project demonstrates the complete data analytics workflow, including:

- Data Collection
- Data Cleaning
- SQL Data Analysis
- KPI Calculation
- Interactive Dashboard Development
- Business Insight Generation

The final dashboard enables business users to monitor sales performance, customer behavior, product demand, and revenue trends for better decision-making.

---

## 🎯 Objectives

- Import sales data into MySQL.
- Clean and organize the dataset.
- Perform business analysis using SQL.
- Build an interactive Power BI dashboard.
- Identify sales trends and customer behavior.
- Generate business insights and recommendations.

## 🔍 Business Insights

### Revenue Analysis

- Generated approximately **₹1.26 Million** in revenue.
- Average order value is approximately **₹1,054**.

### Monthly Sales Trend

Top-performing months:

- June
- May
- January
- March
- February

### Product Performance

Highest revenue-generating products:

- Chair
- Printer
- Laptop
- Tablet
- Monitor

### Payment Analysis

Most preferred payment methods:

- Online Payment
- Cash
- Credit Card
- Debit Card
- Gift Card

Online payment was the most frequently used payment method.
